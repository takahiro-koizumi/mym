== Description ==
Emanon（エマノン）とはno nameの逆さ綴り、つまり「名無し」の意味です。様々な用途に合わせて名前を変えていく（カスタマイズ）できることを目指したビジネスブログ用のテーマです。集客用のCTA機能がテーマの特長です。
