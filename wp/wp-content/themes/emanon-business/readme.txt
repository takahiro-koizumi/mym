== Description ==
Emanon Businessは、簡単な操作で本格的なブログを始めることができるEmanon Pro専用の子テーマです。
Emanon Proの機能に加え、フロントページのデザイン機能を拡張しています。
Emanon Pro本体がインストールされていないと動作しませんのでご注意ください。
